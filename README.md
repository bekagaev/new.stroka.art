# MAYAK — презентация как сайт (deck + интерактив)

Этот проект — **лендинг в формате презентации**:
- каждый слайд = full-screen секция
- скролл с **scroll-snap** (как пролистывание)
- навигация по разделам (клик)
- клавиатура: **↑↓ / PgUp PgDn / Home End**
- пример интерактива: на **слайде 16** кликабельные зоны по персонажам

## Запуск (Windows)

1) Установи Node.js (LTS).
2) В папке проекта:

```bash
npm install
npm run dev
```

Открой: http://localhost:3000

## Где что менять

- Список слайдов и разбиение по разделам: `src/data/deck.ts`
- Основная логика deck/навигации/интерактива: `src/app/page.tsx`
- Глобальные стили (scroll-snap и т.д.): `src/app/globals.css`
- Слайды лежат здесь: `public/slides/slide-01.webp ... slide-38.webp`

## Как обновлять слайды из Figma

1) Экспортируй слайды в PNG (16:9) в хорошем качестве.
2) **Оптимально**: конвертировать PNG → WebP (иначе сайт будет очень тяжёлым).
3) Заменяй файлы в `public/slides/` по именам `slide-XX.webp`.

> Если ты хочешь, чтобы я сделал новую конвертацию/перепаковку — просто кидай ZIP с экспортом.
